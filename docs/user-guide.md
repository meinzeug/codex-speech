# User Guide

## Getting Started
- Launch the app with `codex-stt-assistant`.
- Pick your STT model path in Settings > STT.
- Press Record or use Ctrl+Space to capture speech.

## Basic Usage
- Speak or type in the input panel.
- Use Send to inject text into the Codex terminal.
- Toggle Auto-submit to send STT results immediately.

## Tips
- Use a 16kHz mono microphone for best results.
- Select a smaller model for lower CPU usage.

## Shortcuts
See the Shortcuts dialog inside the app.
